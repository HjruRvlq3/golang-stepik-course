## About the Course

The course is focused on learning the Go (Golang) programming language
and its core features.

During the course, the following topics are covered:
basic language constructs, data structures, working with files and JSON,
as well as fundamentals of concurrent programming.

The course is suitable for anyone interested in learning Golang.

## Topics

- Introduction and language basics
- Functions and data structures
- Working with data and the standard library
- Concurrency
- Additional topics
